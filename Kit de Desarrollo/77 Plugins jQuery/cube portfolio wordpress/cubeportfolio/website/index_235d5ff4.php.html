Main Features

  • Responsive Design

    Cube Portfolio adapts his layout to the viewing device by using a fluid grid. This way you offer a consistent user experiences across all devices.

  • Fully Customizable

    You can customize everytingh from grid layout, lightbox style to every possible animation. With just a little HTML, CSS ans JS you can customize the plugin to fulfill your needs.

  • 6 Starter Templates

    Cube Portfolio includes 6 starter templates: Juicy Project, Lightbox Gallery, Meet the Team, Full Screen Layout, Masonry Layout ans Blog Posts. Choose what is best for you.

  • Captions

    Cube Portfolio has predefined captions, but it is very simple to add your own HMTL formatted containers inside the entries with your own CSS. You can create captions from almost any static HTML elements.

  • CSS3 Animations

    The plugin use the best browser features when available (CSS3 transition and transform, GPU acceleration) and fallback to simple animations (javascript animations) for legacy browsers.

  • Premium Support & Updates For Free

    Connect to our support stuff to get your answers and stay up to date with free updates for life.
